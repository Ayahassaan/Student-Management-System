# Student Management System

A console-based Student Management System developed using C++.

## Features

- Add Student.
- Display Students.
- Search Students by:
    - ID
    - Name
    - Department
- Update Student Information.
- Delete Students.
- Calculate Statistics:
    - Number of Students.
    - Highest GPA Student.
    - Lowest GPA Student.
    - Average GPA.
- Automatic Sorting by Name.
- Save and Load Students using File Handling.
- Input Validation.

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- STL (Vector)
- File Handling
- Searching Algorithms
- Sorting Algorithms
- References
- Lambda Functions
- Input Validation
